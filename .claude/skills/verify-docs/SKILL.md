문서-코드 일치성을 검증하세요.

## 검증 대상

### 1. API 엔드포인트 vs API 문서
- 코드에 정의된 라우트/엔드포인트 수집
- 문서에 기술된 엔드포인트와 비교
- 파라미터, 응답 형식, 상태 코드 일치 확인

### 2. 코드 동작 vs PRD 요구사항
- PRD의 각 요구사항 항목 추출
- 대응하는 코드 구현 존재 여부 확인
- 구현 완전성 평가

### 3. README 설치 절차 vs 실제 설정
- README의 설치 단계 추출
- 실제 필요한 설정/의존성과 비교
- 누락된 단계 식별

## 불일치 분류
| 심각도 | 설명 | 액션 |
|--------|------|------|
| Critical | API 동작과 문서 완전 불일치 | PR 차단 + 이슈 생성 + 즉시 알림 |
| Major | 파라미터/응답 형식 불일치 | PR 경고 + 이슈 생성 + AI 수정 제안 |
| Minor | 오타, 예시 코드 불일치 | PR 코멘트 + AI 문서 업데이트 PR |
| Info | 스타일 차이, 선택적 정보 | 주간 보고서 포함 |

## 결과 형식
```markdown
### 문서-코드 일치성 검증 결과

**전체 일치율**: XX%

**불일치 항목**:
| 심각도 | 문서 위치 | 코드 위치 | 설명 |
|--------|-----------|-----------|------|
| Major | docs/api.md:42 | src/routes.ts:87 | ... |
```
